================================================================================
AFTER SETTING VARIABLES:
================================================================================

1. Click "Save Changes" in Render
2. Render will automatically redeploy your app
3. Check the logs to verify:
   - "Firebase Admin SDK initialized"
   - "Firestore database connected"
   - "Admin user created successfully" (first time)
   - "Server running on port 3001"

4. Update your frontend BASE_URL to point to Render:
   MyExpoApp/config/api.ts
   PRODUCTION_URL = 'https://your-render-app.onrender.com'

5. Test the API:
   https://your-render-app.onrender.com/api/health

================================================================================
IMPORTANT SECURITY NOTES:
================================================================================

1. Never commit .env to git
2. Use Render's environment variables dashboard (encrypted)
3. Change ADMIN_PASSWORD after first login
4. Set ENABLE_ADMIN_BOOTSTRAP=false after first admin is created
5. Use strong JWT_SECRET in production (generate new one)
6. Keep FIREBASE_PRIVATE_KEY secret
